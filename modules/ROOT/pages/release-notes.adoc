*May 2025*

*  The following new modules were developed:
** xref:stories_eEvaluation.adoc[eEvaluation]
** xref:stories_eAwarding.adoc[eAwarding]
** xref:stories_eQualification.adoc[eQualification]

** ePO core
** eNotice
** eCatalogue
** eOrdering
** eFulfilment
** eContract
** eSubmission
** eInvoicing

* The GitHub issues justifying these updates can be found https://github.com/OP-TED/ePO/milestone/14[here]
