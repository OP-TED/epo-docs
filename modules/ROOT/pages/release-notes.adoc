== Release notes

*  eInvoicing module was released.
*  Updates for the following modules were made:
** ePO core
** eCatalogue
** eOrdering
** eFulfilment
** eNotice
** eSubmission
* The (closed) github issues justifying these updates can be found https://github.com/OP-TED/ePO/milestone/10[here]
