== Release notes


*  eAccess module was released.
*  eSubmission module was released.
*  Updates for the following modules were made:
** ePO core
** eCatalogue
** eOrdering
** eFulfilment
** eContract
** eNotice.
* The (closed) github issues justifying these updates can be found https://github.com/OP-TED/ePO/milestone/9[here]
