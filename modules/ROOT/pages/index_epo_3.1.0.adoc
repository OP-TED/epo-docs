= *eProcurement Ontology Version 3.1.0*

== Latest Production Release: 16.12.2022

== *Release Overview*

== *Changes in V3.1.0*
* a new module, eOrdering, was introduced
* the inclusion of notice systematisation in the eNotice module
* changes to Procurement Object: classes and attributes (Procedure, PlannedProcurementPart, Lot)
* a restructure of the Roles Hierarchy
* updates to model2owl (uml2owl):
** a combined glossary
** the provision of turtle output files
** the implementation of a metadata management mechanism
* a revision and labelling of GitHub issues 
* GitHub issue fixes for the Q4 2022 milestone
* Updates for standard forms mappings - TED-RDF-mapping
