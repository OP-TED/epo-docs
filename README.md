# eProcurement Ontology - Working Group Documents

This branch is used for documenting the work of the eProcurement Ontology Working Group. 

The eProcurement Ontology (ePO) provides the formal, semantic foundation for the creation and reuse of linked open data in the domain of public procurement in the EU. It is developed by the Publications Office of the European Union. 

To read the ePO documentation please visit: https://docs.ted.europa.eu/
